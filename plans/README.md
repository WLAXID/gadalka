## Конвенция

- Каждая фаза — отдельный файл `phase-N-<slug>.md`
- Документ переживает фазу: после её завершения сюда дописывается `phase-N-findings.md` с фактами
- Изменения плана — отдельным коммитом, в стиле `docs(plans): ...`
- Решения по go/no-go между фазами фиксируются явно: какие gate-критерии выполнены, какие — нет
